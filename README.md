# WineAdvisor - backend

Backend of application advising on wine purposes based on data provided by client

#### Technology stack:

* Java 8
* Spring Boot
* Spring Data
* Spring Security
* MongoDB
* MySQL
* Logback
* Weka
* Apache POI

#### To run this app:
1) Install Java
2) Install Maven
3) Install MongoDB
4) Install MySQL
5) Start databases

then you can start application with:

```
mvn spring-boot:run
```
